# 气球砰砰砰 (Balloon Pop)

## 游戏玩法说明

### 游戏简介
气球砰砰砰是一款轻松有趣的消除类游戏，玩家需要通过点击或触摸屏幕来消除气球，获得高分。

### 基本操作
- **鼠标控制**：点击屏幕上的气球来消除它们
- **触摸控制**：在移动设备上，通过触摸屏幕点击气球进行消除
- **游戏目标**：尽可能多地消除气球，收集分数

### 游戏规则
- 点击气球即可消除并得分
- 随着游戏进行，难度会逐渐增加
- 游戏自动记录最高分
- 挑战自己的最高分记录

### 如何开始游戏
1. 在浏览器中打开 `index.html` 文件
2. 等待游戏加载完成
3. 点击屏幕开始游戏

### 游戏特色
- 简单易上手的操作方式
- 适合所有年龄段的玩家
- 完全免费的游戏体验
- 支持PC和移动设备
- 流畅的动画效果

### 技术要求
- 支持HTML5的现代浏览器
- 对于移动设备，建议使用Chrome或Safari浏览器
- 确保设备有足够的性能支持游戏运行
